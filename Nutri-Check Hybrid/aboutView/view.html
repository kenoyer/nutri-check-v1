Nutri-Check

Version 1

The application takes input in the form of “the name of the food item and quantity” consumed by the user either via manual text entry or by voice. The food item is checked with a centralized nutrition database over the Internet that has calorific values for a large number of food items as well as nutritional values of each food item. The application provides the amount of calories consumed with nutrition-wise split of the food consumed. Your application offers a nutrition check report and offers some recommended advice supported by medical references. Developed by Kenoye R.
